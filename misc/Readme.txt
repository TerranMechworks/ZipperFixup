Installation
============

1. Copy `zippatch.exe` and `zipfixup.dll` to your game directory.
2. Run `zippatch.exe`. This creates a patched copy of the game named `<Game name>fixup.exe`.
3. Run the new fixup executable.

Uninstallation
==============

1. Delete `zippatch.exe`, `zipfixup.dll`, and `<Game name>fixup.exe`.
